Gallery

Here are some colorful pictures of different flows that I have simulated in the past. The EllipSys CFD code was used for most of the simulations.

Asymmetric diffuser:

  • 2D RANS simulation (\(k\)-\(\omega\)-SST)
  • Setup based on Buice & Eaton (1997).

Square duct:

  • 3D RANS simulation (WJ-EARSM)
  • Only the lower-left quadrant of the duct is simulated.
  • EARSM is able to predict the secondary flow in the duct (not possible for linear EVMs).

Actuator line:

  • 3D URANS simulation (\(k\)-\(\omega\)-SST)
  • Uniform inflow
